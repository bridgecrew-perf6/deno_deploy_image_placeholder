# deno_deploy_image_placeholder

SVG 图片占位图

![](https://imgph.deno.dev)

**使用方式**

```
https://imgph.deno.dev?w=[width]&h=[height]?t=[文字]&bg=[背景颜色]&fg=[文字颜色]
```

示例：

- https://imgph.deno.dev/?w=300&h=100&t=占位符

  ![](https://imgph.deno.dev/?w=300&h=100&t=占位符)

- https://imgph.deno.dev/?w=300&h=100&bg=000&fg=FFF

  ![](https://imgph.deno.dev/?w=300&h=100&bg=000&fg=FFF)
